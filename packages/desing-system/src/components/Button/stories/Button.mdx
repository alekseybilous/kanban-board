# Button

Interactive element that triggers actions when clicked. Essential for forms, dialogs, and user interactions.

## Overview

<Canvas of={ButtonStories.Default} />

<Controls sort="requiredFirst" />

## Usage

```tsx
import { Button } from '@kanban-board/design-system';

function Example() {
  return (
    <Button
      variant="primary"
      size="medium"
      onClick={() => console.log('Clicked!')}
    >
      Click me
    </Button>
  );
}
```
