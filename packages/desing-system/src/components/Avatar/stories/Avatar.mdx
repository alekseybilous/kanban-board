# Avatar

A profile picture component built with Radix UI primitives, featuring automatic fallbacks and customizable sizes and shapes.

## Overview

<Canvas of={AvatarStories.Default} />

<Controls sort="requiredFirst" />

## Usage

```tsx
import { Avatar } from '@kanban-board/design-system';

// Simple usage
<Avatar
  src="https://example.com/avatar.jpg"
  alt="John Doe"
  fallback="JD"
  size="medium"
  variant="default"
/>;

// Using primitives for more control
import {
  AvatarRoot,
  AvatarImage,
  AvatarFallback,
} from '@kanban-board/design-system';

<AvatarRoot size="large" variant="rounded">
  <AvatarImage src="https://example.com/avatar.jpg" alt="John Doe" />
  <AvatarFallback>JD</AvatarFallback>
</AvatarRoot>;
```

## Components

### Avatar

A convenient wrapper component that combines all avatar primitives for simple usage.

### AvatarRoot

The root container that manages the avatar context and provides size/variant to child components.

### AvatarImage

The image element that displays the user's profile picture. Automatically falls back to AvatarFallback if the image fails to load.

### AvatarFallback

The fallback content displayed when the image is unavailable or fails to load. Typically shows user initials.
