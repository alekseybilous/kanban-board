# Select

A flexible dropdown component for selecting options from a list, built with Radix UI primitives and custom styling.

## Overview

<Canvas of={SelectStories.Default} />

<Controls sort="requiredFirst" />

## Usage

```tsx
import {
  SelectRoot,
  SelectTrigger,
  SelectValue,
  SelectContent,
  SelectItem,
} from '@kanban-board/design-system';

<SelectRoot>
  <SelectTrigger>
    <SelectValue placeholder="Select an option..." />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="option1">Option 1</SelectItem>
    <SelectItem value="option2">Option 2</SelectItem>
    <SelectItem value="option3">Option 3</SelectItem>
  </SelectContent>
</SelectRoot>;
```

## Components

### SelectRoot

The root container that manages the select state and provides context to child components.

<ArgTypes
  of={SelectStories.Default}
  include={['size', 'variant', 'value', 'onValueChange', 'disabled']}
/>

### SelectTrigger

The clickable trigger that opens the dropdown and displays the selected value.

### SelectValue

Displays the selected value or placeholder text within the trigger.

### SelectContent

The dropdown container that holds all selectable options.

### SelectItem

Individual selectable options within the dropdown.

### SelectSeparator

Visual separator for grouping related options.

### SelectLabel

Label for grouping options within the dropdown.
