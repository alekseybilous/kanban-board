# Dropdown

A versatile dropdown menu component built with Radix UI primitives, supporting various interaction patterns including checkboxes, radio groups, and submenus.

## Overview

<Canvas of={DropdownStories.Default} story={{ height: '300px' }} />

<Controls sort="requiredFirst" />

## Usage

```tsx
import {
  DropdownRoot,
  DropdownTrigger,
  DropdownContent,
  DropdownItem,
  DropdownSeparator,
} from '@kanban-board/design-system';

<DropdownRoot>
  <DropdownTrigger>Open Menu</DropdownTrigger>
  <DropdownContent>
    <DropdownItem>Profile</DropdownItem>
    <DropdownItem>Settings</DropdownItem>
    <DropdownSeparator />
    <DropdownItem>Logout</DropdownItem>
  </DropdownContent>
</DropdownRoot>;
```

## Components

### DropdownRoot

The root container that manages the dropdown state and provides context to child components.

### DropdownTrigger

The clickable element that opens the dropdown menu.

### DropdownContent

The dropdown container that holds all menu items and content.

### DropdownItem

A clickable menu item for standard actions.

### DropdownCheckboxItem

A menu item with checkbox functionality for toggling options.

### DropdownRadioGroup

Container for radio button menu items that allows single selection.

### DropdownRadioItem

A radio button menu item for single selection within a group.

### DropdownLabel

A non-interactive label for grouping related menu items.

### DropdownSeparator

A visual separator for organizing menu items into sections.

### DropdownSub

Container for creating nested submenus.

### DropdownSubTrigger

A menu item that opens a submenu when hovered or focused.

### DropdownSubContent

The content container for submenu items.
