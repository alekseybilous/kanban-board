## 💡 Usage Guidelines

### ✅ Do's

- Use the 4px grid system consistently
- Prefer semantic tokens (xs, sm, md, lg, xl) for better maintainability
- Scale spacing proportionally across breakpoints
- Maintain consistent spacing between similar elements

### ❌ Don'ts

- Avoid arbitrary pixel values outside the scale
- Don't mix different spacing approaches in the same component
- Avoid too many different spacing values in one layout
- Don't forget about touch target minimums (44px)

### 🎨 Best Practices

```css
/* Button spacing */
.button {
  padding: var(--space-sm) var(--space-md); /* 8px 16px */
}

/* Card layout */
.card {
  padding: var(--space-lg); /* 24px */
  gap: var(--space-md); /* 16px between elements */
}

/* Page sections */
.section {
  margin-bottom: var(--space-xl); /* 32px */
}
```
