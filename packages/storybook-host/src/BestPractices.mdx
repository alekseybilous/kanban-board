# Best Practices

Follow these guidelines to build consistent, maintainable, and high-quality user interfaces with our design system.

## ğŸ¨ Design Token Usage

### âœ… Do: Use Semantic Tokens

```css

  background-color: var(--surface);
  border: 1px solid var(--border);
  color: var(--text);
```

### âŒ Don't: Use Raw Color Values

```css

  background-color: #ffffff;
```

### Token Selection Guide

- **Text**: Use `--text`, `--text-secondary`, `--text-tertiary`
- **Backgrounds**: Use `--surface`, `--canvas` variants
- **Borders**: Use `--border`, `--border-subtle`, `--border-strong`
- **Actions**: Use `--primary-*`, `--secondary-*` tokens
- **States**: Use `--success-*`, `--warning-*`, `--danger-*` tokens

```tsx
// âœ… Good - Detecting system preference
function useSystemTheme() {
  const [theme, setTheme] = useState(() => {
    if (typeof window !== 'undefined') {
      return window.matchMedia('(prefers-color-scheme: dark)').matches
        ? 'dark'
        : 'light';
    }
    return 'light';
  });

  useEffect(() => {
    const media = window.matchMedia('(prefers-color-scheme: dark)');
    const listener = (e) => setTheme(e.matches ? 'dark' : 'light');
    media.addEventListener('change', listener);
    return () => media.removeEventListener('change', listener);
  }, []);

  return theme;
}
```

## ğŸ¯ State Management

### âœ… Do: Handle Loading States

```tsx
// âœ… Good - Clear loading indicators
function SubmitButton({ onSubmit }) {
  const [loading, setLoading] = useState(false);

  const handleSubmit = async () => {
    setLoading(true);
    try {
      await onSubmit();
    } finally {
      setLoading(false);
    }
  };

  return (
    <Button onClick={handleSubmit} loading={loading} disabled={loading}>
      {loading ? 'Saving...' : 'Save Changes'}
    </Button>
  );
}
```

### âœ… Do: Provide Feedback

```tsx
// âœ… Good - User feedback for actions
function DeleteButton({ onDelete, itemName }) {
  const [status, setStatus] = useState('idle');

  const handleDelete = async () => {
    if (status === 'confirming') {
      setStatus('deleting');
      await onDelete();
      setStatus('deleted');
      setTimeout(() => setStatus('idle'), 2000);
    } else {
      setStatus('confirming');
    }
  };

  const getButtonProps = () => {
    switch (status) {
      case 'confirming':
        return { variant: 'destructive', children: 'Confirm Delete?' };
      case 'deleting':
        return {
          variant: 'destructive',
          loading: true,
          children: 'Deleting...',
        };
      case 'deleted':
        return { variant: 'success', children: 'Deleted!' };
      default:
        return { variant: 'destructive', children: 'Delete' };
    }
  };

  return <Button {...getButtonProps()} onClick={handleDelete} />;
}
```
